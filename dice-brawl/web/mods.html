Dice Brawl – Mod (JSON)

Umieść plik moda w folderze „Pobrane/Downloads”, a w grze kliknij przycisk „MOD” (lub naciśnij M), aby go załadować.

Format

  • weaponCost: liczba – koszt broni w sklepie
  • startCoins: liczba – startowa liczba monet
  • playerColor: tablica [R,G,B] – kolor gracza (0–255)

Przykład

{
  "weaponCost": 150,
  "startCoins": 50,
  "playerColor": [255, 200, 0]
}

Jak używać

  1. Zapisz powyższy JSON jako np. mój-mod.json w Pobrane/Downloads.
  2. Uruchom grę i kliknij „MOD” (lub naciśnij M), wybierz plik.
  3. W sklepie (B) zobaczysz nowy koszt broni. HUD pokaże nazwę załadowanego moda.
